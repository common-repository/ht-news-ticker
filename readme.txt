A Breaking news ticker based on JQuery ,Displays a sliding list of post titles. you can use it with a simple shortcode.

== Description ==

<b>HT News Ticker</b> plugin show sliding breaking news on your website and you can use it with a simple shortcode.

Check official website for live demo : http://happythemez.com/demo/plugin/ht-news-ticker/

**Features You can display:**

*   Number of Posts to display.

[Live Demo](http://happythemez.com/demo/plugin/ht-news-ticker/)


**How to Use**<br>
Just use <b>`[ht_ticker]`</b> short code.<br>

== Installation ==

= From your WordPress dashboard =

1. Visit **Plugins > Add New**
2. Search for **HT News Ticker**
3. Activate **HT News Ticker** from your Plugins page.
4. After all the things done copy this <b>`[ht_ticker]`</b> short code  and paste it within your post/page.
5. If you want news ticker in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates.

= From FTP =

1. Download **HT News Ticker**.
2. Upload the **HT News Ticker** directory to your **/wp-content/plugins/** directory.
3. Activate **HT News Ticker** from your Plugins page.
4. After all the things done copy this <b>`[ht_ticker]`</b> short code  and paste it within your post/page.
5. If you want news ticker in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates.

= Shortcodes =

<b>Default</b>
<pre>[ht_ticker]</pre>

<b>Title Styles (Text headline , Font color, Bg color)</b>
<pre>[ht_ticker id="3" Title="Latest News" tbgcolor="#353739" tcolor="#fff"]</pre>

<b>Box styles (Background Color and Link Color)</b>
<pre>[ht_ticker id="2" bgcolor="#07BBD8" linkcolor="#fff"]</pre>

<b>Box styles (height and max rows)</b>
<pre>[ht_ticker id="4" rowheight="60" maxrow="2"]</pre>

<b>Speed contral</b>
<pre>[ht_ticker id="5" speed="200"]</pre>

<b>Duration</b>
<pre>[ht_ticker id="6" duration="2500"]</pre>

<b>Total Posts</b>
<pre>[ht_ticker id="7" numberposts="2"]</pre>
